Out the Closet

Out the Closet was a project for the Design for the Web class at the University of San Francisco. The directions of the project was to create a fictional benefit event that donates to an organization. Out the Closet is a 3-day music festival set in Golden Gate Park, San Francisco. It takes place during Pride Month from dates June 15-17. The festival features queer artists. All funds will be donated to LGBTQ Task Force.The project included physical merch.
